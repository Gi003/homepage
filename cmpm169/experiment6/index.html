Experiment 5 -Space through Words

Description

For this project I knew I wanted to make an interactive story art piece. I wanted to make it ambitiously large, with over 20 locations in the game. For this I decided I wasnt going to hard code any of the playability and I learned how to deal with JSON parsing in js. I also wanted to make a cool anima

Technical

Overall I created a literary story device by making two classes which would add or delete buttons and text and other DOM elements according to what was read from a JSON file. I added around 15 locations to this game.
